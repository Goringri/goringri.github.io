# JINMAN's ClassTool 🏫

초등학교 교실에서 활용할 수 있는 교육용 웹 도구 모음입니다.

## 📚 프로젝트 소개

학생들의 수업 참여를 높이고 학습 효과를 극대화하기 위해 제작된 교육용 웹 애플리케이션입니다. 별도의 설치 없이 브라우저에서 바로 실행할 수 있으며, 직관적인 UI로 누구나 쉽게 사용할 수 있습니다.

## 🎮 주요 기능

### 1. 소수의 나눗셈 (Math Challenge)
- **파일**: `math_game.html`
- **설명**: 소수의 나눗셈 연산 연습 게임
- **특징**:
  - 초급/중급/고급 난이도 선택
  - 60초 시간 제한 모드
  - 명예의 전당 시스템 (로컬 저장)
  - 감성적인 글래스모피즘 디자인

### 2. 딥러닝 게임 (AI Deep Learning)
- **파일**: `deeplearning.html`
- **설명**: AI 학습 원리를 게임으로 체험
- **특징**:
  - 동물 분류 체험 (강아지 vs 고양이)
  - 특징 추출(Feature Extraction) 시각화
  - 로봇 레벨업 시스템
  - 귀여운 이모지 기반 UI

### 3. 자리 배치 (Seating Chart)
- **파일**: `seating_chart.html`
- **설명**: 학급 모둠 자리 배치 도구
- **특징**:
  - 4명 1조 모둠 구성
  - 드래그 앤 드롭으로 자리 교환
  - 랜덤 배정 기능
  - 학생 수 자유 설정

### 4. 단어 깜빡이 (Word Flash)
- **파일**: `word_flash.html`
- **설명**: 영어 단어 학습 플래시카드
- **특징**:
  - 최대 10개 단어 세트 입력
  - 4가지 학습 모드:
    1. 단어 학습 (영어+뜻)
    2. 뜻 말하기 (영어만 표시)
    3. 영어 말하기 (뜻만 표시)
    4. 랜덤 게임
  - 자동 넘김 속도 조절
  - 진행률 표시

## 🚀 사용 방법

1. **다운로드**
   ```bash
   git clone [repository-url]
   ```

2. **실행**
   - `index.html` 파일을 웹 브라우저로 열기
   - 또는 각 도구 파일을 개별적으로 실행

3. **필요 환경**
   - 모던 웹 브라우저 (Chrome, Firefox, Safari, Edge 등)
   - 인터넷 연결 (구글 폰트 로딩용)

## 📂 파일 구조

```
JINMAN's ClassTool/
├── index.html              # 메인 화면 (도구 선택)
├── math_game.html          # 소수의 나눗셈 게임
├── deeplearning.html       # 딥러닝 체험 게임
├── seating_chart.html      # 자리 배치 도구
├── word_flash.html         # 단어 깜빡이 도구
└── README.md               # 프로젝트 설명서
```

## 💡 기술 스택

- **HTML5**: 구조 및 콘텐츠
- **CSS3**: 스타일링 및 애니메이션
  - Glassmorphism (유리 질감 디자인)
  - 리퀴드 애니메이션
  - 반응형 레이아웃
- **Vanilla JavaScript**: 모든 인터랙션 구현
- **Google Fonts**: 한글/영문 폰트

## 🎨 디자인 특징

- **메인 화면**: 리퀴드 애니메이션 + 글래스모피즘 효과
- **수학 게임**: 따뜻한 파스텔 톤의 감성 디자인
- **딥러닝 게임**: 귀여운 이모지 + 명확한 정보 전달
- **자리 배치**: 실제 교실 느낌의 직관적 레이아웃
- **단어 깜빡이**: 칠판 콘셉트의 교실 분위기

## 📱 반응형 지원

- 데스크톱, 태블릿, 모바일 환경 지원
- 16:9 비율 최적화 (프로젝터 수업 활용)
- 가로 모드 권장

## 🔧 커스터마이징

```css
:root {
    --primary: #4a69bd;     /* 기본 색상 */
    --bg: #f1f2f6;          /* 배경색 */
    /* ... */
}
```
